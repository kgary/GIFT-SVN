Did you know...

The learner module in GIFT contains information about the learner's traits and states including performance, engagement, and emotions.
The domain module in GIFT contains information for the learner, feedback for the learner, and a model of how an expert would perform this task.
The pedagogical module in GIFT contains generalized learning strategies based on learner states.
The tutor-user interface (TUI) is the window in GIFT that provides written (and sometimes verbal) feedback to the learner.
The GIFT Event Reporting Tool (ERT) can support output in CSV format for various types of research designs.
The GIFT Forums provide an easy to access information source on how to resolve existing problems as well as to query the community for further help.
GIFT includes a set of common surveys to use in courses, or as examples for new surveys.
The GIFT Users Symposium is the annual meeting of the community to discuss their experiences with GIFT.
GIFT stands for Generalized Intelligent Framework for Tutoring
GIFT is a framework, which means it can be extended to accommodate many different types of tutors
