You are about to enter a medic tutorial scenario.

In this tutorial, data is collected on the casualty shown above. GIFT will monitor the casualty's blood volume and will adjust the learner's anxiety state using a RapidMiner data model.

As time progresses, the casualty's blood volume will decrease, theoretically affecting the learner's anxiety level. Applying a tourniquet to the casualty will fix the blood volume, causing the learner's anxiety state to become constant. 

The scenario will end automatically after several state changes.
Advanced Users: To view Learner State changes, use the Learner State Panel located in the GIFT Monitor application.
