This folder contains domain knowledge used by GIFT.  

Domain knowledge includes, but is not limited to:
	- courses
	- DKFs
	- training material (e.g. pdfs, URLs, images)
	- scenario files (plus training application support files such as models, terrains and scripts)
	
The folder structure is as follows:
Exports/
    Contains GIFT export zip files.
Imports/
	Contains GIFT import zip files.  Import files are GIFT exports that are being imported into this version of GIFT.
runtime/
    Contains course folders that were executed during runtime.
workspace/
    Contains the location of authored course folders.
workspace/Public
    This is where publicly released courses live.
workspace/templates
	Contains various authoring template and example files.
workspace/<username>
    There is where user authored courses live.
