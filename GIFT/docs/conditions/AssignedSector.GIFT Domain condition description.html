Assigned Sector Condition

This condition checks whether the provided members are looking in their assigned sectors. The author can define a maxium offset of degrees from the perfect viewing of the sector that is allowed.  Also, there is an optional free look duration that can be provided to allow momentary scanning outside of the assigned sector (default is 0 seconds).

Assessment Evaluations (Real-time)

  • Above Expectation
  • At Expectation
    • case: All assessed members are looking at the assigned sector within the provided angle threshold.
  • Below Expectation
    • case: At least one assessed member is looking outside of their assigned sector beyond the free look duration.
  • Unknown
    • case: The default initial value of this condition.

Possible Scored Evaluations (overall)

  • counts the number of violations (going from At to Below Expecation)
  • sums the duration of time in Below Expectation

Miscellaneous details

  • this condition does not complete

Intended Training Applications and Domains

  • any application that can produce entity state messages

Consumed State Messages (used for assessment)

  • GIFT Entity State
