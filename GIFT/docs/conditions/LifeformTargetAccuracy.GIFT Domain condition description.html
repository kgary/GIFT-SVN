Lifeform Target Accuracy Condition

This condition checks whether the learner shoots every target with two shots (i.e. double tap) and doesn't miss with any shots.  The starting locations of all the targets must be provided as an input to this condition. 

Assessment Evaluations (Real-time)

    Default: 

  • Above Expectation
  • At Expectation
    • case: the default value
    • case: each time a target is hit the second or subsequent times
  • Below Expectation
    • case: each time the learner fires their weapon and it doesn't hit a pre defined target
  • Unknown

    Customize:

  • count and time: an event happens if a learner's shot misses an identified target from the list.

Possible Scored Evaluations (overall)

  • counts the number of times the learner misses a pre defined target, i.e. number of below expectation assessment

Miscellaneous details

  • the condition does not complete
  • the condition can't be repeated

Intended Training Applications and Domains

  • any application that uses the DIS GIFT Gateway introp plugin
  • any application that creates the necessary messages

Consumed State Messages (used for assessment)

  • GIFT Entity State
  • GIFT Detonation
