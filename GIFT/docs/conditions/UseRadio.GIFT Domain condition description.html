Use Radio Condition

This condition is used to determine when the learner indicates that the radio should be used.  Currently the GIFT learner action radio button is something that the GIFT Tutor webpage can provide if the author enables it.  This learner action, along with other learner actions, can be used to provide another mechanism for GIFT to capture learner actions which is useful in situations where you are unable to incorporate certain events/tools into the external application but need them as part of your assessment.

Assessment Evaluations (Real-time)

  • Above Expectation
  • At Expectation
    • case: the learner tutor action message indicates that the use radio is the action taken
  • Below Expectation
  • Unknown
    • case: the default value

Possible Scored Evaluations (overall)

  • counts the number of times the learner tutor action message indicates that the use radio is the action taken
  • provides the amount of time it took the learner to cause the use radio action to take place

Miscellaneous details

  • the condition completes once the learner tutor action message indicates that the use radio is the action taken
  • the condition can be repeated

Intended Training Applications and Domains

  • normally the GIFT Tutor exposes the learner action buttons (if selected during authoring) but external applications can also produce this message

Consumed State Messages (used for assessment)

  • GIFT Learner Tutor Action
