Marksmanship Precision Condition

This condition checks whether the expected number of shots were received and if so, calculates the mean distance (dispersion) of the shots from the center of the shot group.  The expected number of shots is required as input. 

The expected dispersion distance is 0.5 meters.

Assessment Evaluations (Real-time)

  • Above Expectation
  • At Expectation
    • case: the default value
    • case: the dispersion is less than or equal to expected
  • Below Expectation
    • case: the dispersion is greater than expected
  • Unknown

Possible Scored Evaluations (overall)

  • counts the number of times when the expected number of shots is reached.  This can happen multiple times if the current number of shots provided by the application is reset.
  • provides the amount of time it took to get to the expected number of shots

Miscellaneous details

  • the condition completes when the expected number of shots is reached
  • the condition can be repeated as long as the current number of shots is reset at some point

Intended Training Applications and Domains
