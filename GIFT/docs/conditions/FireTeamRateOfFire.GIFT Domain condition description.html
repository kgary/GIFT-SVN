Fire Team Rate of Fire Condition

This condition checks whether the provided members are maintaining an appropriate rate of fire while this condition is being assessed. The assessment uses a formula called Talking the Guns (TG) which is

TG ratio = 1 - (total dead space time / total assessment time)

The total assessment time is provide by the author and is a sliding window.  Assessments happen after the first 30 seconds and then every 5 seconds after that.  Both of these values can be changed by the author.  Also, the TG ratio required to achieve Below/At/Above expectation can be defined.

Assessment Evaluations (Real-time)

  • Above Expectation
    • case: when the TG ratio is above the upper bounds of the At Expectation TG ratio.
  • At Expectation
    • case: when the TG ratio is above the upper bounds of the Below Expectation TG ratio.
  • Below Expectation
    • case: when the TG ratio is below the upper bounds of the Below Expectation TG ratio.
  • Unknown
    • case: The default initial value of this condition.

Possible Scored Evaluations (overall)

  • counts the number of violations (going from Above/At to Below Expecation)
  • sums the duration of time in Below Expectation

Miscellaneous details

  • this condition does not complete

Intended Training Applications and Domains

  • any application that can produce weapon fire messages

Consumed State Messages (used for assessment)

  • GIFT Weapon Fire
