Request External Attribute Condition

This condition queries an external training application for a variable value associated with an entity in the scenario.

E.g. can be used to retrieve the target 'down' animation state in a gunnery table exercise in VBS.

Assessment Evaluations (Real-time)

  • Above Expectation
  • At Expectation
  • Below Expectation
  • Unknown
    • case: The default initial value of this condition.

Possible Scored Evaluations (overall)

  • none

Miscellaneous details

  • this condition does not complete

Intended Training Applications and Domains

  • none

Consumed State Messages (used for assessment)

  • Variables State Result
