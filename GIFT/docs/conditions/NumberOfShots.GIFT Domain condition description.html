Number of Shots Fired Condition

This condition checkes if the expected number of shots were fired.

Assessment Evaluations (Real-time)

  • Above Expectation
  • At Expectation
    • case: the learner fires the expected number of shots, no less, no more
  • Below Expectation
    • case: the default value
    • case: if the number of shots fired does NOT equal the expected number
  • Unknown

Possible Scored Evaluations (overall)

  • count - the number of shots fired
  • time - the time it takes to fire the expected number of shots

Miscellaneous details

  • the condition completes once the number of expected shots is reached
  • the condition can not be repeated

Intended Training Applications and Domains

  • Marksmanship
  • any application that can produce GIFT Rifle Shot messages

Consumed State Messages (used for assessment)

  • GIFT Rifle Shot
