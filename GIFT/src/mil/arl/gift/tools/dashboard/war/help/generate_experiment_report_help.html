Generating Reports with GIFT Local
Need to generate frequent or offline reports? Try GIFT Local in Desktop Mode!

GIFT Local's Event Report Tool (ERT) also provides more options in how reports can be generated.
How Do I Generate Reports with GIFT Local?
Step 1: Download GIFT Local .
Step 2: Install GIFT Local.
Step 3: Download your experiment's subject data
Step 4: Unzip the subject data into the GIFT/output/logger/messages folder within your GIFT Local installation.
Step 5: Launch the Event Report Tool (ERT) from GIFT/scripts/launchControlPanel.bat
For further instructions on how to use the Event Report Tool, please consult the Event Report Tool Instructions.
