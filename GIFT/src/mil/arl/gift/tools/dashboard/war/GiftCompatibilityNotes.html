GIFT Compatiblity Notes for Internet Explorer

 

Please note that GIFT uses “Font Awesome” icons for display in the web browser.  In order for the web browser to render these icons properly, Internet Explorer must have the “Font Download” setting set to “Enabled” under the Security settings.

If you are experiencing issues with missing icons,

·         Option 1:  Change the “Font Download” setting to “Enabled” under the Internet Explorer Security settings.

·         Option 2:  Try an alternative browser such as Firefox or Chrome.

·         Option 3:  If running GIFT in standalone DESKTOP mode, change the url to:  http://localhost:8080/dashboard.
